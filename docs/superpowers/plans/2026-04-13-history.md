# Rip History Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add SQLite-backed rip history tracking every disc scan and rip session, with episode continuation, duplicate detection, CLI subcommand interface, and TUI overlay.

**Architecture:** `src/history.rs` module containing `HistoryDb` struct wrapping `rusqlite::Connection`. `Option<HistoryDb>` lives on `App` (TUI) and is passed into CLI functions, following the same pattern as `Config`. Each thread opens its own DB connection (WAL mode for safe concurrency). CLI history subcommand uses argv pre-check before clap parsing.

**Tech Stack:** rusqlite (bundled SQLite), uuid (batch_id generation), chrono (already in deps for timestamps)

**Spec:** `docs/superpowers/specs/2026-04-13-history-design.md`

```rust
impl HistoryDb {
    pub fn open(path: &Path) -> Result<Self> {
        if let Some(parent) = path.parent() {
            std::fs::create_dir_all(parent)
                .with_context(|| format!("creating history DB directory: {}", parent.display()))?;
        }
        let conn = rusqlite::Connection::open(path)
            .with_context(|| format!("opening history DB: {}", path.display()))?;
        let db = Self { conn };
        db.init()?;
        Ok(db)
    }

    pub fn open_memory() -> Result<Self> {
        let conn = rusqlite::Connection::open_in_memory()?;
        let db = Self { conn };
        db.init()?;
        Ok(db)
    }

    fn init(&self) -> Result<()> {
        // Enable WAL mode (no-op for in-memory)
        let _: String = self.conn.query_row("PRAGMA journal_mode=WAL", [], |r| r.get(0))?;
        // Enable foreign keys
        self.conn.execute_batch("PRAGMA foreign_keys = ON")?;
        self.run_migrations()?;
        self.cleanup_stale_sessions()?;
        Ok(())
    }
}
```

The `run_migrations()` method checks for `schema_version` table, creates it if missing, reads current version, and applies any pending migration SQL. Migration 1 creates all three tables and indexes from the spec schema.

The `cleanup_stale_sessions()` method finds `in_progress` sessions older than 4 hours and marks them as `failed`.

Add to `impl HistoryDb`:

`start_session` — INSERTs into sessions with `started_at` = current ISO 8601 timestamp, returns `last_insert_rowid()`.

`finish_session` — UPDATEs sessions SET `status`, `finished_at` = now WHERE `id`.

`record_disc_playlists` — INSERTs each playlist into disc_playlists (use a transaction for atomicity).

`record_file` — uses `INSERT INTO ripped_files ... ON CONFLICT(session_id, playlist) DO UPDATE SET ...` UPSERT. Returns `id` of the inserted/updated row via `SELECT id FROM ripped_files WHERE session_id = ? AND playlist = ?` after the upsert.

`update_file_status` — UPDATEs ripped_files SET `status`, `finished_at` = now, optionally `error` WHERE `id`.

Helper function for current timestamp:
```rust
fn now_iso8601() -> String {
    let now = std::time::SystemTime::now()
        .duration_since(std::time::UNIX_EPOCH)
        .unwrap()
        .as_secs();
    // Format as ISO 8601 using chrono
    chrono::DateTime::from_timestamp(now as i64, 0)
        .unwrap()
        .format("%Y-%m-%dT%H:%M:%S")
        .to_string()
}
```

```rust
use anyhow::{bail, Result};
use regex::Regex;

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum ParsedDuration {
    Relative(u32),          // days
    Absolute(String),       // ISO date string "YYYY-MM-DD"
}

pub fn parse_duration(s: &str) -> Result<ParsedDuration> {
    let s = s.trim();
    if s.is_empty() {
        bail!("empty duration string");
    }

    // Try relative: Nd, Nmonth(s), Nyear(s)
    let re = Regex::new(r"^(\d+)(d|months?|years?)$").unwrap();
    if let Some(caps) = re.captures(s) {
        let n: u32 = caps[1].parse()?;
        if n == 0 {
            bail!("duration must be > 0");
        }
        let unit = &caps[2];
        let days = if unit == "d" {
            n
        } else if unit.starts_with("month") {
            n * 30
        } else {
            n * 365
        };
        return Ok(ParsedDuration::Relative(days));
    }

    // Try absolute: YYYY-MM-DD
    let date_re = Regex::new(r"^\d{4}-\d{2}-\d{2}$").unwrap();
    if date_re.is_match(s) {
        return Ok(ParsedDuration::Absolute(s.to_string()));
    }

    bail!("invalid duration: '{}' (expected Nd, Nmonths, Nyears, or YYYY-MM-DD)", s)
}

impl ParsedDuration {
    /// Convert to an ISO 8601 cutoff timestamp for DB queries.
    pub fn to_cutoff_date(&self) -> Result<String> {
        match self {
            ParsedDuration::Relative(days) => {
                let secs = *days as i64 * 86400;
                let now = std::time::SystemTime::now()
                    .duration_since(std::time::UNIX_EPOCH)
                    .unwrap()
                    .as_secs() as i64;
                let cutoff = chrono::DateTime::from_timestamp(now - secs, 0).unwrap();
                Ok(cutoff.format("%Y-%m-%dT%H:%M:%S").to_string())
            }
            ParsedDuration::Absolute(date) => {
                Ok(format!("{}T00:00:00", date))
            }
        }
    }
}
```

```rust
/// Resolve the history DB path: BLUBACK_HISTORY_PATH env → config → XDG default.
pub fn resolve_db_path(config: Option<&crate::config::Config>) -> std::path::PathBuf {
    if let Ok(path) = std::env::var("BLUBACK_HISTORY_PATH") {
        return std::path::PathBuf::from(path);
    }
    if let Some(path) = config.and_then(|c| c.history_path()) {
        return std::path::PathBuf::from(path);
    }
    let data_dir = std::env::var("XDG_DATA_HOME")
        .map(std::path::PathBuf::from)
        .unwrap_or_else(|_| {
            let home = std::env::var("HOME").unwrap_or_else(|_| "/tmp".into());
            std::path::PathBuf::from(home).join(".local/share")
        });
    data_dir.join("bluback").join("history.db")
}
```

```rust
#[test]
fn test_history_list_show_stats_lifecycle() {
    let db_path = temp_db();
    let _ = std::fs::remove_file(&db_path);

    // List on empty DB
    let out = bluback()
        .args(["history", "list"])
        .env("BLUBACK_HISTORY_PATH", &db_path)
        .output()
        .unwrap();
    assert!(out.status.success());

    // Stats on empty DB
    let out = bluback()
        .args(["history", "stats"])
        .env("BLUBACK_HISTORY_PATH", &db_path)

        .unwrap();
    assert!(out.status.success());
    let stdout = String::from_utf8_lossy(&out.stdout);

    // Cleanup
```
